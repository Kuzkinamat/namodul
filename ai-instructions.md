Инструкция для разработки терминала/бота проверки торговых стратегий

Технологический стек: HTML, JavaScript, Python

Библиотека TradingView Lightweight Charts v5.1.0 (использовать синтаксис именно этой версии)

Написание/правка кода выполняется только по команде "давай", после уточнения деталей задачи

Не изменять и не удалять код, не имеющий отношения к текущей задаче.

При добавлении нового функционала обязательно добавлять вывод логов для отладки: ответы сервера, фрагменты данных, ключевые этапы выполнения…

Итоговый функционал системы:

1. Выбор рабочего диапазона: 

   1 день/1 неделя/1 месяц/ 1 год, по умолчанию 1 месяц.

2. Выбор таймфрейма:

   1m / 5м / 15м, по умолчанию 1м.

3. Выбор источника:

   Для онлайн источников фильтр - основные пары Forex. При выборе источника - подгрузка доступных пар/файлов в список пар.

4. Выбор пары/файла/тикера:

   При выборе пары рассчитать количество свечей для загрузки: \[количество минут в диапазоне\] / \[таймфрейм в минутах\]
