# FOREX strategy tester / trading signal bot

Веб-приложение для тестирования торговых стратегий на форекс с использованием библиотеки Lightweight Charts.

## Возможности

- Загрузка данных из Twelve Data API (реальные котировки)
- Загрузка локальных данных из CSV файлов (офлайн-режим)
- Отображение свечных графиков с индикаторами (SMA, Bollinger Bands, Stochastic, MACD)
- Тестирование стратегий на исторических данных
- Визуализация сигналов и графика баланса
- Гибкая настройка диапазона и таймфрейма

## Использование

1. Откройте `index.html` в браузере.
2. Выберите источник данных:
   - **Twelve Data API** – требуется API-ключ (уже встроен в код)
   - **Local JS** – локальные данные, сконвертированные из CSV
3. Выберите валютную пару, диапазон и таймфрейм.
4. Включите индикаторы и стратегии через меню.
5. Запустите тестирование стратегии.

## Локальные данные (CSV → JS)

Для работы с собственными историческими данными используйте скрипт `csv-to-js.js`.

### Формат CSV

Файл должен содержать строки в формате:
```
YYYY-MM-DD HH:MM,open,high,low,close,volume
```
Пример:
```
2025-06-30 22:15,1.17841,1.17874,1.17834,1.17871,181
```

### Конвертация

1. Поместите CSV файл(ы) в корневую директорию проекта.
2. Запустите скрипт:
   ```bash
   node csv-to-js.js
   ```
   Скрипт автоматически найдет все CSV файлы и создаст соответствующие JS файлы (например, `EURUSD_M5_data.js`).

3. Подключите полученный JS файл в `index.html` (скрипт уже добавлен автоматически для файла `EURUSD_M5_data.js`).

### Автоматическая загрузка

При открытии приложения, если обнаружен хотя бы один JS файл с данными (например, `EURUSD_M5_data.js`), приложение автоматически:
- Установит источник **Local JS**
- Выберет соответствующий таймфрейм (из имени файла, например, `_M5` → 5 минут)
- Загрузит данные и отобразит график.

### Добавление новых данных

Чтобы добавить новый CSV файл, просто скопируйте его в проект и запустите скрипт конвертации. После обновления страницы данные появятся в выпадающем списке пар (отфильтрованы по таймфрейму).

## Структура проекта

- `index.html` – основной интерфейс
- `main.js` – логика приложения, управление графиком, провайдеры данных
- `prov-twelvedata.js` – провайдер данных Twelve Data API
- `ind.js` – расчет индикаторов (Stochastic, MACD, EMA)
- `strategy.js` – тестовая стратегия и расчет PnL
- `csv-to-js.js` – скрипт конвертации CSV в JS
- `EURUSD_M5.csv` – пример исторических данных (5-минутные свечи EUR/USD)
- `EURUSD_M5_data.js` – сконвертированные данные (генерируется автоматически)
- `lightweight-charts.standalone.production.js` – библиотека графиков (внешняя)

## Зависимости

- Lightweight Charts v4.0.0 (включена в проект)
- Современный браузер с поддержкой ES6

## Примечания

- Twelve Data API имеет лимиты на количество запросов. Для бесплатного тарифа – 800 запросов в день.
- Локальные данные не требуют интернет-соединения и подходят для тестирования на больших исторических периодах.
- Все вычисления индикаторов выполняются на стороне клиента.
